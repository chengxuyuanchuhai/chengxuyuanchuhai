## UncleLeon心声
UncleLeon认为，23-24年大模型AI技术的狂飙猛进带来了新的机遇，个人能力因AI得到了急剧放大（我个人目前的体会至少是10倍放大），以前独立开发者不能做的事情现在可以做了。走出去面向全球市场，我认为是存在一些机会的，也能提高自己的能力和视野。加上现在不健康的职场环境，至少尝试下还是值得的。
